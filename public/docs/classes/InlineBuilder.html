    Class InlineBuilder

    Builder for constructing inline Serchat markdown.

    Methods append text to a buffer and return this for chaining. Call build to get the final string.

    Example

    const text = new InlineBuilder()
      .text('Hello, ')
      .bold('world')
      .text('!')
      .build();
    // => 'Hello, **world**!'
