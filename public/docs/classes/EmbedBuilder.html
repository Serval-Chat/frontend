    Class EmbedBuilder

    Builder for constructing rich embed objects.

    Example

    const embed = new EmbedBuilder()
      .setTitle('Hello!')
      .setDescription('This is a rich embed.')
      .setColor(0x5865f2);

    await client.sendMessage(serverId, channelId, embed);
