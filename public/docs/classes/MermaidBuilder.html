    Class MermaidBuilder

    Builder for constructing Mermaid diagrams.

    Example

    const diagram = new MermaidBuilder('graph TD')
      .node('A', 'Start')
      .node('B', 'Stop')
      .edge('A', 'B', 'Go to')
      .build();
